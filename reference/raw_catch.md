# Get raw catch data for a fishery

Get raw catch data for a fishery

## Usage

``` r
raw_catch(fishery, start_date = NULL, end_date = NULL)
```

## Arguments

- fishery:

  Fishery name

- start_date:

  Optional. First date of window.

- end_date:

  Optional. Last date of window.

## Value

Table of raw, unexpanded catch reported from angler interviews.
